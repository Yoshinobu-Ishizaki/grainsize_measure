## はじめに

C2600P について grainsize_measure ツールで解析した二つの試料を比較する。

- **加工なし**（c2600p_asis）：未熱処理の原板(O材)
- **焼鈍**（c2600p_600c60min）：600°C・60分焼鈍後に空冷

再結晶温度以上での加熱により結晶粒界が移動し（粒成長），
粒数が減少して平均粒径が増大することが予想される。

------------------------------------------------------------------------

## 結論

焼鈍処理によって，結晶粒径は顕著に増大した。

- **粒数**：加工なし 713 粒 → 焼鈍 131 粒（約 82% 減少）
- **平均等価円直径**：加工なし 28.2 µm → 焼鈍 64.8 µm（約 2.3 倍）
- **中央値**：加工なし 26 µm → 焼鈍 46.5 µm
- **標準偏差**：加工なし 16.7 µm → 焼鈍 44.8 µm

粒径分布は両条件とも右裾の長い非対称分布を示した。 AIC
最小の最適モデルは，加工なしが **ガンマ分布**，焼鈍が **対数正規分布**
であった。

Hall-Petch 則によれば，粒径の増大は降伏応力の低下（軟化）をもたらし，
延性・加工性の向上が期待される。
600°C・60分焼鈍材は，プレス成形など塑性加工前の中間焼鈍として
適切な軟化状態にあると考えられる。

## データと解析手法

### 解析パラメータ

各試料の画像解析に用いたパラメータを以下に示す。

| パラメータ                    |   加工なし   | 600°C/60分焼鈍 |
|:------------------------------|:------------:|:--------------:|
| スケール (px/µm)              |     0.49     |      0.49      |
| 検出手法                      | color_region |  color_region  |
| CLAHE クリップ上限            |      0       |       5        |
| 適応閾値ブロックサイズ        |      23      |       15       |
| モルフォロジー閉演算半径 (px) |      1       |       3        |
| 最小粒面積 (px²)              |      5       |       50       |
| 最小特徴サイズ (px)           |      9       |       50       |
| 境界粒子除外                  |     TRUE     |      TRUE      |

主な差異：

- **CLAHE クリップ上限**：焼鈍材ではコントラスト強調（clip =
  5.0）を適用。粒界コントラストが低い大粒に対応。
- **モルフォロジー閉演算半径**：焼鈍材で大きく設定（1 → 3
  px）し，太くなった粒界を確実に閉じる。
- **最小粒面積 /
  最小特徴サイズ**：粒が粗大化しているため，ノイズ除去の閾値を引き上げ（9
  → 50 px²）。
- **適応閾値ブロックサイズ**：各画像の輝度分布に合わせて調整（23 → 15
  px）。

## 記述統計

| 統計量         | 加工なし | 600°C/60分焼鈍 |
|:---------------|---------:|---------------:|
| 粒数           |   713.00 |         131.00 |
| 平均径 (µm)    |    28.21 |          64.78 |
| 中央値 (µm)    |    25.95 |          46.46 |
| 標準偏差 (µm)  |    16.72 |          44.80 |
| 変動係数 (%)   |    59.27 |          69.15 |
| 最小径 (µm)    |     5.15 |          16.45 |
| 最大径 (µm)    |    96.99 |         213.23 |
| 平均面積 (µm²) |   844.31 |        4860.35 |

焼鈍材は粒数が大幅に減少（713 → 131 粒）し， 平均粒径は約 2.3
倍に増大した。 変動係数は両条件とも 100%
前後と大きく，粒径分布の散らばりが顕著である。

## 統計分布フィッティング

> **注記：ベータ分布について** ベータ分布のサポートは閉区間 \[0, 1\]
> に限定されるため， µm 単位の粒径データには適用できない。本解析では
> **対数正規分布・ワイブル分布・ガンマ分布** の 3 種類を対象とする。

各分布の最尤推定を `fitdistrplus::fitdist` で実施した。

### フィッティングパラメータと適合度

| 条件           | 分布         | パラメータ1      | パラメータ2     |    AIC |    BIC |
|:---------------|:-------------|:-----------------|:----------------|-------:|-------:|
| 加工なし       | 対数正規分布 | meanlog = 3.1401 | sdlog = 0.6749  | 5944.5 | 5953.6 |
| 加工なし       | ワイブル分布 | shape = 1.7693   | scale = 31.7751 | 5898.3 | 5907.4 |
| 加工なし       | ガンマ分布   | shape = 2.6591   | rate = 0.0942   | 5896.4 | 5905.5 |
| 600°C/60分焼鈍 | 対数正規分布 | meanlog = 3.9505 | sdlog = 0.6622  | 1302.8 | 1308.5 |
| 600°C/60分焼鈍 | ワイブル分布 | shape = 1.5694   | scale = 72.7244 | 1321.2 | 1327.0 |
| 600°C/60分焼鈍 | ガンマ分布   | shape = 2.4212   | rate = 0.0374   | 1312.8 | 1318.6 |

AIC が最小のモデル（最適フィット）：加工なし = **ガンマ分布**， 焼鈍 =
**対数正規分布**。
